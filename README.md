# -📌 Face Recognition Attendance System
📖 Описание проекта
Данный проект представляет собой систему автоматизированного учёта посещаемости учащихся с использованием технологии распознавания лиц.
Система предназначена для применения в образовательных учреждениях и позволяет: автоматически фиксировать присутствие учеников; определять опоздавших; формировать отчёты для преподавателей и администрации.

🎯 Цель проекта: Разработка программного решения для повышения точности и скорости учёта посещаемости учащихся за счёт использования технологий компьютерного зрения.

⚙️ Основной функционал:

📷 Захват изображения с камеры

🧠 Распознавание лиц учащихся

📊 Формирование списков:

Присутствующих

отсутствующих

опоздавших

📩 Генерация отчёта (например, для директора)

🗂 Поддержка нескольких классов (например, 11Г)

🧩 технологии

Python

OpenCV

Библиотеки распознавания лиц (face_recognition / dlib)
NumPy

📊 Пример работы

После запуска система:

Считывает изображение с камеры

Сравнивает лица с базой данных

Определяет статус ученика

Отправляет отчёт

📈 Результаты

Сокращение времени учёта посещаемости, повышение точности данных, сключение человеческого фактора
